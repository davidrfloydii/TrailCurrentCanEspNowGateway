# TrailCurrentCanEspNowGateway
This repository holds the code, schematics, and CAD files for an ESPNow gateway that relays CAN messages over the ESPNow protocol

# Overview
The reason to relay messages from the CAN Bus to ESPNow is to facilitate a remote device that can go into a tow vehicle. This allows for data being relayed over the CAN bus in a trailer to be displaye remotely and reliably in a vehicle.
